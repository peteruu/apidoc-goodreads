    Introduction

    Welcome to the Goodreads Unofficial API! You can use our API to access Goodreads content. In examples we are using Symfony component called The HttpClient Component.

    Authentication

    Pass your API-key in all API requests as a header. To authorize, use this code:

    Request:

    <?php
$api_key = YOURAPIKEY;
$query_parameters = QUERYPARAMETERS;

$httpClient = HttpClient::create([
    'headers' => ['X-AUTH-API' => $api_key],
]);

$response = $httpClient->request('GET', 'https://admin.bunny-holding.com/api', ['query' => $query_parameters,]);

?>

    Make sure to replace YOURAPIKEY with your API key and replace QUERYPARAMETERS with parameters as it is defined in API.

    Each project has different API key. Before making any request with your api key please be sure you configured allowed titles in admin.

    To make any request use api url: https://admin.bunny-holding.com/api and send us GET request.

    You must replace YOURAPIKEY with your API key from your project in [admin](https://admin.bunny-holding.com).

    Query Parameters

    Query parameters are used to generated specific response. You can combine it in any way you want. Every response is JSON formated.
